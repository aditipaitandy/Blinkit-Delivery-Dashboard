📊 Blinkit / Swiggy Instamart Delivery Performance Dashboard
🔹 Project Overview

This project analyzes delivery operations for quick commerce platforms like Blinkit and Swiggy Instamart using Power BI.

🔹 Key KPIs
Total Orders
Average Delivery Time
Late Delivery %
🔹 Dashboard Features
Peak hour analysis (Orders by Hour)
City-wise delivery performance
Area-wise delay analysis
Delivery status distribution
Revenue analysis
Rider performance evaluation
🔹 Tools Used
Power BI
Excel
🔹 Key Insights
Identified peak ordering hours
Detected delay-prone areas
Compared delivery efficiency across cities
Evaluated rider performance
🔹 Business Impact

This dashboard helps improve delivery efficiency, reduce delays, and optimize resource allocation.
