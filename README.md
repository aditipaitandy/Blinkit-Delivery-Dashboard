# 📊 Blinkit / Swiggy Instamart Delivery Performance Dashboard

## 🔹 Project Overview
Developed an interactive Power BI dashboard to analyze delivery operations for quick commerce platforms such as Blinkit and Swiggy Instamart. The project focuses on evaluating delivery efficiency, identifying delays, and deriving actionable insights to improve operational performance.

---

## 🔹 Key KPIs
- Total Orders  
- Average Delivery Time  
- Late Delivery Percentage  

---

## 🔹 Dashboard Features
- **Peak Hour Analysis** – Identifies high-demand time periods using order trends  
- **City-wise Performance** – Compares delivery efficiency across different cities  
- **Area-wise Delay Analysis** – Highlights delay-prone locations for targeted improvements  
- **Delivery Status Distribution** – Visualizes proportion of on-time vs delayed deliveries  
- **Revenue Analysis** – Examines revenue contribution across cities  
- **Rider Performance Evaluation** – Assesses delivery partner efficiency  

---

## 🔹 Tools & Technologies
- Power BI  
- Microsoft Excel  

---

## 🔹 Key Insights
- Peak order volume is concentrated during specific hours, indicating demand spikes  
- Certain areas consistently exhibit higher delays, suggesting operational inefficiencies  
- Delivery performance varies across cities, highlighting scope for optimization  
- Rider performance analysis helps identify high-performing and underperforming partners  

---

## 🔹 Business Impact
This dashboard enables data-driven decision-making by providing insights into delivery performance, helping organizations optimize resource allocation, reduce delays, and enhance overall customer experience.
